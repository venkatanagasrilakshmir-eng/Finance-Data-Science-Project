# Finance Data Science Project

## Overview
This project analyzes stock market data and predicts stock closing prices using machine learning.

## Features
- Stock market data collection
- Data preprocessing
- Trend analysis
- Machine learning prediction
- Data visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- yfinance

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Machine Learning
6. Visualization

- Stock prediction model
